# 📚 AI 智能学习助手

> **基于艾宾浩斯遗忘曲线的个性化学习规划系统**

一款面向备考学生的 AI 驱动学习工具，通过智能分析题库、识别薄弱知识点、生成个性化学习计划，帮助用户高效备考。

---

## 🎯 产品定位

### 目标用户
- 期末考试备考学生
- 考研/考公等长期备考人群
- 希望提高学习效率的终身学习者

### 核心痛点
| 用户痛点 | 解决方案 |
|---------|---------|
| 不知道重点在哪里 | AI 分析题库，自动提取知识点并评估重要性 |
| 学了就忘，效率低 | 基于艾宾浩斯遗忘曲线的智能复习提醒 |
| 时间分配不合理 | 根据掌握程度智能分配学习时间 |
| 缺乏针对性练习 | AI 生成针对薄弱项的诊断测试 |

---

## ✨ 核心功能

### 1️⃣ 智能题库分析
上传历年真题 / 模拟试卷，AI 自动：
- 📊 提取核心知识点并建立知识图谱
- ⭐ 评估各知识点考察频率与重要程度
- 🎯 识别高频考点与难点

### 2️⃣ 薄弱点诊断系统
- 🧪 AI 生成个性化诊断测试题
- 📈 实时更新知识点掌握度
- 🔍 精准定位知识盲区

### 3️⃣ 艾宾浩斯复习调度
基于科学的遗忘曲线理论，自动安排 7 轮复习：

```
首次学习 → 1天后 → 2天后 → 4天后 → 7天后 → 15天后 → 30天后
```

### 4️⃣ AI 一对一辅导
- 💬 针对薄弱知识点提供苏格拉底式教学
- 📖 生成定制化学习材料
- 🤔 解答疑难问题

## 🚀 快速开始

```bash
# 克隆项目
git clone https://github.com/qd-maker/study-agent.git

# 进入目录
cd study-agent/study-planner

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

访问 http://localhost:3000 开始使用。

### 配置 AI
1. 进入 **设置** 页面
2. 选择 AI 服务商 (OpenAI / Claude / 通义千问 / DeepSeek)
3. 填入 API Key
4. 开始智能学习！

## 🧠 设计思考

### 为什么选择艾宾浩斯曲线？
相比自由复习，科学的间隔重复可以：
- 减少 **60%** 无效重复时间
- 提升 **40%** 长期记忆留存率

### 为什么用 AI 分析题库？
传统方式需要手动整理知识点，AI 可以：
- 快速处理大量非结构化数据
- 自动识别知识点关联关系
- 持续迭代优化识别准确率
